## üöÄ Getting Started

### Installation
1. Install the extension from the [Chrome Web Store](https://chromewebstore.google.com/detail/whisk-automator-bulk-ai-i/fggolibbgjecabhcigloidnnbhfokmkf).
<img src="assest/1-300x200.png" width="800" height="400" alt="App Screenshot">

2. Pin the icon to your Chrome toolbar.
3. Open [Google Labs Whisk](https://labs.google/whisk).

### Setup (Important)
To allow the **Auto-Download** feature to work without pop-ups:
1. Go to Chrome **Settings**.
2. Search for **Downloads**.
3. Toggle **OFF** "Ask where to save each file before downloading."

## üõ† How to Use
1. **Open the Side Panel:** Click the Whisk Automator icon while on the Whisk website.
2. **Add Prompts:** Type or paste your prompts box(one per line).
    - (Optional) Use advanced options to set theme, creativity level, and prompt count.
3. **Configure References:** If using reference images, upload them in the Whisk interface first, then link them in the Automator.
4. **Run:** Click **Start Generating**. The extension will now cycle through your list automatically.
5. **Download:** The tool will handle the generation process automatically.

## üìù Tips for Best Results
* **Delay:** Set a delay of at least **15 seconds** to allow Whisk to process the previous image.
* **Keep Active:** Keep the Whisk tab in focus to ensure the automation scripts run at full speed.
* **Prompt Length:** Try to keep prompts descriptive but concise for better AI accuracy.
